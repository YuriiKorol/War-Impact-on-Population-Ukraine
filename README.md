This project analyzes the number of people who left Ukraine during the war, using Power BI. 
The data was obtained from an open source, initially processed in Excel, and then imported into Power BI. 
Data transformation was done using Power Query, reshaping the data from a wide format to a long format.
I created an interactive dashboard with a map, charts, and KPIs, using DAX formulas to calculate displacement percentages. 
I created a separate parameter, which I used in a DAX formula for dynamic calculations. 
I also encountered an issue where, when applying a Top N filter, the chart did not recalculate percentages based on the overall total shown in the Total card. 
Additionally, I faced challenges with incorrect data types that did not display properly on the charts. 
I worked on renaming and managing measures and parameters to ensure a correct structure for analysis. 
This project showcases my skills in cleaning, transforming data, and building interactive visualizations in Power BI.
